
Menu
===
A directive for creating a menu with multiple items  
  
### Parameters
**mac-menu-item**  
Type: `Expression`  
**mac-menu-select**  
Type: `Function`  
Callback on select  
  
**mac-menu-style**  
Type: `Object`  
Styles apply to the menu  
  
**mac-menu-index**  
Type: `Expression`  
Index of selected item
